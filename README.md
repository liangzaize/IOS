####程序介绍
-《显卡佬》主要的服务领域在计算机硬件，本软件集成了包括各类电脑硬件的参数、电脑硬件方面新闻、用户论坛等功能，用户可以在程序中看到所有电脑硬件比如显卡、CPU等的参数，并且可以浏览最新的与电脑有关的新闻。
